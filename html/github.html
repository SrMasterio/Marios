Funcionalidad GitLens


GitLens es una extensión de Visual Studio Code que mejora la integración con Git, permitiendo visualizar el historial de cambios de un archivo, ver quién modificó cada línea de código y cuándo, comparar versiones y explorar commits y ramas. Facilita la navegación y comprensión del historial de código directamente en el editor, mejorando la productividad.
